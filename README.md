# jwt
Mediante una aplicación WebAssembly (lado cliente puro) se muestran experimentos que permiten entender y establecer el comportamiento de la expiración o caducidad de la validez del JSON Web Token (JWT) para autenticación y autorización del acceso a cada método web expuesto por HTTP en una WebApi (lado servidor puro),  con código en lenguaje C#.
